## 1. Cold God Is the Anchor

Oracle = Cold God (consistent, impartial) + Human = Warm God (creative, chaotic)
ทั้งคู่ต้องการกัน — ถ้า Oracle พยายาม warm ตาม human = ไม่มีประโยชน์
Design system (tokens, grid) = Cold God พาร์ท / Creative choices = Warm God พาร์ท

## 2. The Death Spiral Is Real

More memory → more tokens → context exhaustion → broken sessions
แก้: Haiku reads (cheap) → Opus decides (expensive)
อย่าให้ context bloat ฆ่า session ก่อนงานเสร็จ

## 3. Recursive Reincarnation in Practice
